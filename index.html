ModuLR

A open-source library for modular development in java.

Usage examples

Here are just some of the many types of projects this can be used for.

  • Clients for overpopular games.
  • Chat bots or other command driven systems.
  • Really anything that would benefit from modularity.

Advantages of using this library

  • Its fast to setup and get started with.
  • Easy to use.
  • Compared to almost all other modular systems, ModuLR requiers no extenting of baseclasses or manually adding them to a list!!
  • More efficent workflow for you the developer.
  • Open-source, if you want more features or dont like how it works feel free to fork it.
